# Automated YouTube Testing with Selenium WebDriver

This project provides a suite of automated tests for YouTube, focusing on end-to-end testing of various features like video search, playback, subscribing to channels,
and liking videos using Selenium WebDriver and Python. It's designed to demonstrate the capabilities of Selenium for web automation and testing purposes.

## Features

- **Login Automation**: Automates logging into YouTube.
- **Search Functionality Testing**: Verifies that users can search for videos and receive relevant results.
- **Video Playback Testing**: Ensures that videos play, pause, and seek correctly.
- **Channel Subscription Testing**: Automates subscribing to a YouTube channel and verifies the subscription.
- **Like Video Testing**: Automates the process of liking a video and verifies that the like has been successfully registered.

## Getting Started

### Prerequisites

- Python 3.6 or higher
- Selenium WebDriver
- A webdriver (e.g., ChromeDriver) compatible with your browser version

### Installation

1. Clone this repository to your local machine using `git clone https://github.com/<username>/<repository>.git`.
2. Navigate to the cloned directory.
3. Install the required Python packages with `pip install -r requirements.txt`.
